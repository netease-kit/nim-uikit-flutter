# NimChatKitUI ChangeLog

## 1.2.0(Nov 8, 2023)
### New Features
* 新增@功能
* 新增标记列表
* 位置消息接入方式变更
* 图片消息支持gif格式

### Bug Fixes
* 修复信息同步问题
